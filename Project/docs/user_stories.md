As a student, I want to have a list of lost items publically available, so that I can easily look for my lost items.

As a student, I want the list of lost items to be centralized across the whole campus, so that I don't need to check several different lost-and-found places on campus to look for my items.

As a student, I want the list of lost items to be handled by campus staff, so that security is guaranteed (i.e. my items can't be claimed by random people)

As a student, I want each item to have associated text descriptions so that I can search through items to find a lost item more quickly.

As a lost-and-found staff member, I want an online database of lost items to be widely known to students, so that lost items submitted can be claimed more easily.

As a lost-and-found staff member, I want the process of posting lost items to be fast and simple, so that I can efficiently post lost items on the platform.

As a lost-and-found staff member, I want to keep track of the identity (name, WatIAM) of people who claim the items, so that disputes over claimed items can be resolved and items can be returned to their correct owner.
